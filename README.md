# React-Router-Store-Typescript

## Pre install

* IDE - VSCode
  * install `plugin` - stylint
  * install `plugin` - tslint
* install redux-dev-tools in **Chrome Store**

## Deps

* css
  * css-modules
  * stylus
  * post-css
* react
  * typescript
  * react-router
  * rematch / a framework of redux
* test
  * jest
* webpack v4

## How to use it?

```bash
git clone <repo>
cd <repo>
npm install
```
